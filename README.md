## Description:

- Password generate that take command line arguments to generate passwords
- Generated Password are printed to the terminal screen and rated on its Strength
- Automatically these passwords are saved the clipboard
- Optionally by typing **yes** after generating the password it can be saved to a csv file.

> [!NOTE]
> The csv file should be temporary just to import into a password manager and should be immediately deleted after used.

- Additionally you can import other password-csv files from Google,Firefox, etc into one csv file and import that in your password manger

I had a bit too much fun when making this generator so please use the help function to see the different using available for this password manager

## Journey:

I decided for my project to make a password generator due to my recent interest in cybersecurity. 

This project is partially inspired by [Proton-Pass](https://proton.me/pass) (My current password manager).
When navigating through the app I saw you can import csv files from firefox, google, etc.
There is  also a feature to randomly generate passwords based on specific criteria.

Initially the project was written using the random module as its randomness backbone. After some research I released the random module as pseudorandom and was not fully secure.
The random module was then switched to the secrets module to ensure that these were truly secure and random passwords.

Writing the csv functions I would say was the most difficult portion of this exercise for me.
I released just I was not that comfortable yet with the csv/open module syntax
noticed when I tried to implement ideas I had in my head but they were not working in my code.

For instance **password.csv** is the file I use to store the data generated by the user. 
Initially this function was completely writing all the data and I soon realised I needed to append the data not write it.

I wanted this generator to work via command line arguments so I used the sys module. 
I also used regular expressions to search for the arguments. 
I also wanted to command line arguments to be somewhat modular where users can enter -h or -help to get the same result.

After making the password generator and the csv files. I wanted to do a bit more and wanted there to be some presentation.
Using the tabulate library I added a function to present the csv file in a nice table for the user.
Alongside that I added the options to clear the table for the user.
